# Glossary

**Terms, Expressions and Acronyms (TEA) must be organized alphabetically.**

| **_TEA_** (EN)         | **_TEA_** (PT)               | **_Description_** (EN)                                                                                  |
|:------------------------|:-----------------------------|:--------------------------------------------------------------------------------------------------------|
| **ADDRESS**             | **MORADA**                   | A full postal address, including street, city, postal code, and country.                                |
| **ADMINISTRATOR**       | **ADMINISTRADOR**            | System user with permission to manage physicians, appointments, and view patient data.                  |
| **API**                 | **API**                      | Abbreviation for Application Programming Interface.                                                     |
| **APPOINTMENT**         | **CONSULTA**                 | Represents a scheduled meeting between a physician and a patient.                                       |
| **APPOINTMENT RECORD**  | **REGISTO DE CONSULTA**      | Clinical record created by a physician during an appointment, includes diagnosis and prescriptions.     |
| **CONSULTATION TYPE**   | **TIPO DE CONSULTA**         | Indicates whether the appointment is a first-time or a follow-up consultation.                          |
| **CONTACT INFO**        | **INFORMAÇÃO DE CONTACTO**   | Collection of emails, phone numbers and addresses associated with a person.                             |
| **DEPARTMENT**          | **DEPARTAMENTO**             | Organizational unit grouping physicians by medical area (e.g., Cardiology).                             |
| **DTO**                 | **DTO**                      | Data Transfer Object used to transfer data between API and internal layers without exposing the domain. |
| **HEALTH CONCERN**      | **PROBLEMA DE SAÚDE**        | Medical condition reported by the patient, including treatment and resolution status.                   |
| **INSURANCE INFO**      | **INFORMAÇÃO DE SEGURO**     | Patient’s insurance details: policy number and insurance provider name.                                 |
| **PATIENT**             | **PACIENTE**                 | Person receiving medical care in the clinic.                                                            |
| **PHOTO**               | **FOTOGRAFIA**               | Profile image optionally associated with a user (patient or physician).                                 |
| **PHYSICIAN**           | **MÉDICO**                   | Healthcare professional registered in the system who provides consultations.                            |
| **SPECIALTY**           | **ESPECIALIDADE**            | Medical area of expertise of a physician (e.g., Pediatrics, Neurology).                                 |
| **STATUS**              | **ESTADO**                   | State of an appointment: Scheduled, Cancelled, or Completed.                                            |
| **WORKING HOURS**       | **HORÁRIO DE TRABALHO**      | Time interval that defines the physician's daily availability.                                          |
